```
//
// notes-import.scpt 
// Steven Frank <stevenf@panic.com>
//

//   - Open this file in Script Editor
//   - Run
//   - Select folder containing files to import
//
// Known issues:
//   - Attachments don't get imported
//   - Formatting/HTML is not preserved
//   - No idea how text encoding is handled
```
